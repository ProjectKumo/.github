

# ProjectKumo

**Building the next generation of human-agent interaction.**

GUI-first experiences for people. Terminal-first control surfaces for agents.

## Mission

ProjectKumo exists to build the next generation of human-agent interaction:  
humans get calm, native, GUI-first products; agents get stable, terminal-first  
CLI contracts. Both surfaces share the same control layer so clicking, typing,  
scripting, and delegating stay consistent.
